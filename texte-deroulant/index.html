Ancienne balise marquee

Marquee était une ancienne balise html qui permettait de faire ça.

Maintenant on doit le faire avec un élément after.

Voici comment ça se passe :

Il faut un container et à l'intérieur un élément avec la classe .marquee

    
.marquee-container{
    display:flex;
    align-items: center;
    background:linear-gradient(to right,darksalmon,rebeccapurple);
    color:white;
    overflow:hidden;
    margin-block: 50px;
}

.marquee{
    font-family: sans-serif;
    text-transform: uppercase;
    font-size: clamp(32px,10vw,3.5rem);
    padding-block: 50px;
    animation:marquee 10s infinite linear 1s;
    white-space:nowrap;
}
.marquee:after{
    /* faut que ce soit exactement le même texte */
    content:"LE WEBDESIGN C'EST VRAIMENT TROP COOL.";
}
@keyframes marquee{
    0%{
        transform:translateX(0);
    }
    100%{
        transform:translateX(-50%);
    }
}

Ajouter le after content en JavaScript

Hélas y'a pas de moyen simple de le faire.

https://stackoverflow.com/questions/5041494/selecting-and-manipulating-css-pseudo-elements-such-as-before-and-after-usin

Mais on peut le faire avec un peu trick en html / css : 


<div class="marquee-container-2">
    <div class="marquee-2" data-content="Et en JavaScript ? Ça marche comment ?"></div>
</div>

On met le contenu dans data-content.

En css, on utiliser attr :


.marquee-2:after{
    content:attr(data-content);
}

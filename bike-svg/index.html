Animation Css

Noter l'utiliser de la propriété transform-box.

            
#roueAvant{
    animation:motion 2s infinite linear;
    transform-origin:center;
    transform-box:fill-box;
}

#roueArriere{
    animation:motion 2s infinite linear;
    transform-origin:center;
    transform-box:fill-box;
}

#PedalePP{
    transform-origin:center bottom;
    animation:motion 3s infinite linear;
    transform-box:fill-box;
    backface-visibility: hidden;
}

#PedaleAP{
    transform-origin:center top;
    animation:motion 3s infinite linear;
    transform-box:fill-box;
    backface-visibility: hidden; 
}

#Pedalier{
    animation:motion 3s infinite linear;
    transform-box:fill-box;
    transform-origin:center center;
}

@keyframes motion{
    from{
        transform:rotateZ(360deg);
    }
    to{
        transform:rotateZ(0deg);
    }
}

Ne pas prévoir d'animer des éléments enfants d'élements parents déjà animés. Car c'est totalement broken sous Firefox.

Root EM, comment ça marche ?

            
/* Ou sur html */
:root{
    font-size: 24px;
    line-height: 1.5em;
}

h1, h2, h3, h4, ul, ol {
    margin-bottom: 1.5rem;
}

button {
    display: inline-block;
    padding: 0.75rem 1.5rem;
    font-size:1.2rem;
    /* 24px*1.2 = 28.8px*/
}

p{
    font-size:1rem;
    /* On doit toujours définir pour suivre la taille de root */
}

On définit une taille de base de 24px.

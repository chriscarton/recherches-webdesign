Align items et Justify Items s'appliquent sur la grille. Ces propriétés globales déterminent comment tous les éléments vont se placer dans la grille.

  • start
  • end
  • center
  • stretch (par défaut)
    
#pageItems1 .grid{
    align-items: end;
}
#pageItems2 .grid{
    justify-items: end;
}

La propriété Place Items existe et peut-être utilisée si les deux valeurs correspondent.

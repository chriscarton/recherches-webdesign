Element SVG à inclure dans le Css via url

<svg viewBox="0 0 200 200" xmlns='http://www.w3.org/2000/svg'>
    <filter id='noiseFilter'>
        <feTurbulence 
        type='fractalNoise' 
        baseFrequency='3.65' 
        numOctaves='3' 
        stitchTiles='stitch' />
    </filter>
    
    <rect width='100%' height='100%' filter='url(#noiseFilter)' />
</svg>

Structure HTML

body > container > sometext + div.grainy-gradient

Css

body {
    background-color: chocolate;
}
.grainy-gradient{
    isolation: isolate;
    position:absolute;
    inset:0;
    z-index: -1;
}
.grainy-gradient::before {
    content:' ';
    background:
        linear-gradient(to right, red, transparent),
        url(filter-noise.svg);
    color:white;
    /* MORE NOISE */
    filter: contrast(500%) brightness(1000%);
    position:absolute;
    inset:0; 
    z-index:0; 
}
.grainy-gradient::after {
    content:' '; 
    position:absolute;
    inset:0; 
    z-index:0;
    /* MÊME COULEUR QUE LE FOND */
    background:chocolate;
    mix-blend-mode:multiply;
}

Comment ça marche ?

grainy-gradient::before a un dégradé qui va vers du transparent
body et grainy-gradient::after ont le même background.
grainy-gradient::after a la propriété mix-blend-mode:multiply.

Attention

Plus l'élément couvert par le dégradé est haut plus il faudra augmenter baseFrequency dans le svg

baseFrequency='3.65'
